Clipboard to QR Code

Chrome-Erweiterung zum Konvertieren von Zwischenablage-Text in QR-Codes

Download

Laden Sie die Chrome-Erweiterung herunter, um Texte aus Ihrer Zwischenablage schnell in QR-Codes umzuwandeln.

Erweiterung herunterladen

Installationsanleitung

  1. Laden Sie die Datei clipboard-to-qr-extension.tar.gz herunter
  2. Entpacken Sie das Archiv in einen Ordner auf Ihrem Computer
  3. Öffnen Sie Google Chrome und navigieren Sie zu chrome://extensions/
  4. Aktivieren Sie den "Entwicklermodus" in der oberen rechten Ecke
  5. Klicken Sie auf "Entpackte Erweiterung laden"
  6. Wählen Sie den Ordner aus, in den Sie die Dateien entpackt haben
  7. Die Erweiterung sollte nun in Chrome installiert sein und in der Toolbar erscheinen

Verwendung

  1. Kopieren Sie einen Text in Ihre Zwischenablage
  2. Klicken Sie auf das Erweiterungssymbol in der Chrome-Toolbar
  3. Sie sehen den Inhalt Ihrer Zwischenablage im Popup
  4. Klicken Sie auf "Get QR"
  5. Ein QR-Code wird generiert und automatisch in Ihre Zwischenablage kopiert
  6. Sie können diesen QR-Code nun überall einfügen, wo Bilder erlaubt sind (Word, Paint, usw.)
